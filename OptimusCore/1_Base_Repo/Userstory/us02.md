The following changes are required in ZIP:

LOB – 057

SBU – SP Specialties

Policy Type – AH Accident & Health

Target Market – VIC Vistors to Canada

Locations section - Select the ‘Locations’ option on the ‘Maintain Policy’ screen

Adding a location
o On the ‘Work with Location’ screen, select F9 to add a location
o On the ‘Maintain Location’ screen, under the country field, ADD a new
country code OOC for Out of Canada - A&H only
o When ‘OOC’ is selected, the out of country should prepopulate with the
following address:
Out of Canada - A&H only – Address field
Out of Canada - A&H only - City field
Province and postal code fields are not required

There will only be one location that will be out of Canada, ZIP should not allow for
two locations to be out of Canada

Other locations can be added if they are other provinces, but only one out of
Canada location will be allowed.

The following 3 items should also prepopulate as blank:
o Cresta Zone
o Windstorm Zone
o Flood Zone
